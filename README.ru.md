# Удаление фона

Инструмент для пакетного удаления фона с изображений. Использует [rembg](https://github.com/danielgatis/rembg) для удаления фона, убирает белую окантовку по краям и автоматически обрезает результат.

## Установка (Windows)

### 1. Установите Python

Скачайте и установите Python 3.10+ с [python.org](https://www.python.org/downloads/). При установке отметьте галочку **"Add Python to PATH"**.

### 2. Установите зависимости

Откройте Командную строку или PowerShell и выполните:

```
pip install rembg[gpu] Pillow numpy scipy
```

Если у вас нет GPU, используйте:

```
pip install rembg Pillow numpy scipy
```

### 3. Подготовьте изображения

Поместите исходные изображения (`.png`, `.jpg`, `.jpeg`) в папку `raw_images`.

### 4. Запуск

```
python test.py
```

Обработанные изображения появятся в папке `processed_images`.

## GUI-приложение

`app.py` -- версия с графическим интерфейсом: выбор папок и настройки прямо в окне. Запуск:

```
python app.py
```

## Сборка .exe (конечному пользователю Python не нужен)

### 1. Установите зависимости для сборки

```
pip install pyinstaller rembg Pillow numpy scipy
```

### 2. Скачайте модель (если ещё не скачана)

```
python -c "from rembg import new_session; new_session('isnet-general-use')"
```

### 3. Соберите

```
pyinstaller remove_bg.spec
```

Готовый `RemoveBackground.exe` появится в папке `dist`. В него упакованы Python, все зависимости и модель ИИ -- на целевой машине ничего устанавливать не нужно.

## Настройки

Параметры очистки краёв можно изменить в `test.py` в вызове `clean_white_edges()` или в панели настроек GUI-приложения:

- `border_size` (по умолчанию `3`) -- на сколько пикселей от края проводить очистку. Увеличьте, если белая окантовка остаётся.
- `white_threshold` (по умолчанию `240`) -- порог "белого" цвета. Меньшее значение очищает агрессивнее.
